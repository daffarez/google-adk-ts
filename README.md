# Google ADK (TS)

This repository contains an AI agent built using the **Google Agent Development Kit (ADK)**.

## Installation

1. Install dependencies:

   ```bash
   npm install
   ```

2. Configure Environment:
   Create a `.env` file in the root directory and add your API key:

   ```bash
   GEMINI_API_KEY=your_api_key
   ```

## Run the Agent

1. Terminal

   ```bash
   npm start
   ```

2. Web Interface

   ```bash
   npm run web:interface
   ```

---

Read more at [Agent Development Kit - Google](https://google.github.io/adk-docs/get-started/).
